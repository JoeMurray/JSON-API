NAPI Samples

Informational

  1. get info on all nymi bands in the vicinity
  2. check for authenticated nymi bands in the vicinity
  3. watch continuously for certain events

Authentication

  1. provision one nymi band
  2. register with roaming authenticator
  3. roaming authenticate

Operations on Provisioned Nymi Bands

  1. generate a pseudo random number
  2. buzz
  3. sign a SHA-256 hash
  4. get symmetric keys (any created keys will be unguarded)
  5. get symmetric keys (any created keys will be guarded)
  6. enable TOTP
  7. retrieve TOTP

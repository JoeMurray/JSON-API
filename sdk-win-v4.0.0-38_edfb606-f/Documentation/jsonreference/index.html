Protocol Conventions

JSON keys are strings with the names of the request and response attributes.

The types of the attributes are the types expected in JSON.

In the case of attributes with enum types, the JSON value should be a string containing the name of the enum.

All attributes in a request are required unless a default value is specified, in which case that value is used if the attribute is not in the JSON request.

If an attribute in a response has a default, it will always have that value in the JSON response.

Common Terms

These terms are used frequently in the code. Instead of defining them each time they occur, we have provided them in the list below. You should quickly become familiar with these terms.

  • pid is the provision ID of the Nymi Band
  • tid is the temporary ID of a Nymi Band
  • NBA = Nymi Band App, an application provided by Nymi that manages the user's biometric profile and authentication types
  • NEA = Nymi-Enabled Application, that communicates with Nymi Bands
